# Тест-кейсы для таск-трекера Авито

## Задание: Создание задачи

### TC001: Успешное создание задачи
**Severity:** High  
**Priority:** High  
**Behavior:** Positive  
**Pre-conditions:** Открыт сайт таск-трекера: https://avito-tech-internship-psi.vercel.app/issues

**Test Steps:**
| Step | Action | Expected Result |
|------|--------|-----------------|
| 1 | Нажать кнопку "Создать задачу" | Открывается форма создания задачи с полями: Название, Описание, Проект, Приоритет, Статус, Исполнитель  |
| 2 | Заполнить поле "Название" текстом "Тестовая задача" | Поле заполняется введенным текстом |
| 3 | Заполнить поле "Описание" текстом "Описание тестовой задачи" | Поле заполняется введенным текстом |
| 4 | Нажать на поле "Проект" и выбрать любой из проектов в выпадающем списке | Поле заполняется нужным проектом
| 5 | Нажать на поле "Приоритет" и выбрать любой из приоритетов в выпадающем списке | Поле заполнится нужным приоритетом
| 6 | Оставить поле "Статус" пустым/неактивным | Поле остаётся пустым/неактивным
| 6 | Нажать на поле "Исполнитель" и выбрать любого из исполнителей в выпадающем списке | Поле заполнится нужным исполнителем
| 7 | Нажать кнопку "Создать" | - Форма закрывается<br>- Список задач обновляется <br>- Задача создается и появляется в списке задач |

### TC002: Валидация обязательных полей
**Severity:** Medium  
**Priority:** High  
**Behavior:** Negative  
**Pre-conditions:** Открыта форма создания задачи

**Test Steps:**
| Step | Test Case | Action | Expected Result |
|------|-----------|--------|-----------------|
| 1 | Без названия | - Оставить "Название" пустым<br>- Заполнить все обязательные поля<br>- Нажать "Создать" | Кнопка не должна прожиматься |
| 2 | Без проекта | - Заполнить "Название"<br>- Оставить "Проект" пустым<br>- Заполнить остальные обязательные поля<br>- Нажать "Создать" | Кнопка не должна прожиматься |
| 3 | Без приоритета | - Заполнить "Название" и "Проект"<br>- Оставить "Приоритет" пустым<br>- Заполнить остальные обязательные поля<br>- Нажать "Создать" | Кнопка не должна прожиматься |
| 4 | Без статуса | - Заполнить "Название", "Проект", "Приоритет"<br>- Оставить "Статус" пустым<br>- Заполнить "Исполнитель"<br>- Нажать "Создать" | Задача создается и появляется в списке задач |
| 5 | Без исполнителя | - Заполнить все обязательные поля кроме "Исполнитель"<br>- Оставить "Исполнитель" пустым<br>- Нажать "Создать" | Кнопка не должна прожиматься |
