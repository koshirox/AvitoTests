## О проекте
Набор автоматических тестов для проверки функционала таск-трекера Авито. Тесты написаны на Python с использованием Selenium WebDriver
## Расположение
Тесты находятся в корневой папке в файле Task_Tracker.py <br>
Также в корневой папке находится скачанный chromedriver (v142.0.7444.175)
## Группы тестов
TC001-TC002 - Создание задачи
<br>
TC003-TC012 - Поиск задачи
<br>
TC013-TC018 - Открытие карточки задачи
<br>
TC019-TC024 - Переход на доску проекта

## Использование
Запуск всех тестов: 
python Task_Tracker.py 
<br>
Запуск всех тестов с детальным отчётом: 
python Task_Tracker.py -v
