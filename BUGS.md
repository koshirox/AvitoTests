# Баг-репорты для Таск-трекера Авито

## BUG001: Задачи без описания не отображаются в списке задач после создания

### Основная информация
- **ID бага:** BUG-001
- **Серьезность:** Высокий
- **Приоритет:** Высокий
- **Статус:** Открыт
- **Тип:** Функциональный баг

### Окружение
- **ОС:** Windows 11 24H2
- **Браузер:** Google Chrome 142.0.7444.176
- **URL:** https://avito-tech-internship-psi.vercel.app/index.html
- **Версия:** Test

### Описание
При создании задачи с пустым полем "Описание" задача успешно создается, но не отображается в общем списке задач. Поле "Описание" не является обязательным согласно требованиям.

### Шаги воспроизведения
1. Открыть сайт таск-трекера
2. Нажать кнопку "Создать задачу"
3. Заполнить обязательные поля:
   - Название: "Тестовая задача без описания"
   - Проект: выбрать любой проект
   - Приоритет: выбрать любой приоритет
   - Исполнитель: выбрать любого исполнителя
4. Оставить поле "Описание" пустым
5. Нажать кнопку "Создать"
6. Перейти к списку всех задач

### Ожидаемый результат
Задача создается и отображается в списке задач

### Фактический результат
Задача не отображается в списке задач, хотя форма создания закрывается без ошибок

## BUG002: Фильтр по статусу "Done" показывает задачи со всеми статусами

### Основная информация
- **ID бага:** BUG-002
- **Серьезность:** Средний
- **Приоритет:** Высокий
- **Статус:** Открыт
- **Тип:** Функциональный баг

### Окружение
- **ОС:** Windows 11 24H2
- **Браузер:** Google Chrome 142.0.7444.176
- **URL:** https://avito-tech-internship-psi.vercel.app/index.html
- **Версия:** Test

### Описание
При применении фильтра по статусу "Done" в списке отображаются задачи со всеми статусами, а не только с статусом "Done". Фильтры для других статусов работают корректно.

### Шаги воспроизведения
1. Открыть сайт таск-трекера
2. Нажать на фильтр "Статус"
3. Выбрать статус "Done" из выпадающего списка
4. Наблюдать за отображаемыми задачами

### Ожидаемый результат
В списке отображаются только задачи со статусом "Done"

### Фактический результат
В списке отображаются задачи со всеми статусами (Backlog, InProgress, Done)

## BUG003: Изменение названия задачи в карточке не сохраняется

### Основная информация
- **ID бага:** BUG-003
- **Серьезность:** Высокий
- **Приоритет:** Высокий
- **Статус:** Открыт
- **Тип:** Функциональный баг

### Окружение
- **ОС:** Windows 11 24H2
- **Браузер:** Google Chrome 142.0.7444.176
- **URL:** https://avito-tech-internship-psi.vercel.app/index.html
- **Версия:** Test

### Описание
При редактировании названия задачи в карточке и нажатии кнопки "Обновить" изменения не сохраняются - название остается прежним. Другие поля редактируются корректно.

### Шаги воспроизведения
1. Открыть сайт таск-трекера
2. Найти и открыть любую существующую задачу (например, "Адаптация карточки для мобильных устройств")
3. В поле "Название" изменить текст на новое значение
4. Нажать кнопку "Обновить"
5. Снова открыть ту же задачу

### Ожидаемый результат
Название задачи изменяется на новое значение

### Фактический результат
Название задачи остается прежним, изменения не сохраняются
